SMARTSTRIDE Senior Design Project

Runs with AWS using:
  Amplify
  RDS Database
  Lambda

Pi_Code folder is for the code on the Raspberry Pi that collects data from the SmartStride device
WebsiteCode folder holds all files needed for AWS functionality
  Lambdas is all Lambda codes used
  Docs is a collection of images and documents used in Amplify website
All other files are test data to show data format and to test enviornment
